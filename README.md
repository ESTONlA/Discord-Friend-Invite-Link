# Discord Friend Invites
See the [LICENSE](LICENSE) file for details.

Simple UI tool for creating, viewing, and managing Discord friend invite links directly from the browser.

Created by **Estonia**

---

## Features

• Create new Discord friend invite links
• View all active invites
• Check invite usage and expiration
• Delete all active invites
• Clean draggable UI inside Discord

---

## Preview

A small window appears inside Discord where you can manage your friend invites without digging through menus.

---

## How to Use

1. Open Discord in your browser
2. Press **F12** to open Developer Tools
3. Go to the **Console** tab
4. Paste the script
5. Press **Enter**

A UI window will appear where you can manage your invites.

---

## What the Script Does

The script interacts with the Discord API endpoint:

`/users/@me/invites`

Functions include:

• Creating invite links
• Listing active invites
• Showing invite usage data
• Removing all invites

No external servers are contacted. All requests go directly to Discord.
